# FLAPPYBIRD_VHDL_DE2
Flappy Bird game written completely in VHDL on an Altera DE2 FPGA board. Written by Drew Everson, Brady Heinke, Michael Fisher, and Max Zubic at the University of Wisconsin-Platteville.
